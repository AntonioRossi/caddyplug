caddyplug
=========

caddyplug is [Caddy](https://caddyserver.com) plugin manager using Go plugins.

## Requirements
* Go 1.8
* Linux
* Caddy with [Plugin Loader](http://) [Still WIP].

## Usage
```
  Usage:
    caddyplug [flags] plugins...

  Flags:
    -list      list plugins
    -uninstall uninstall plugins
```

## Note
This is not an official Caddy product.
